Common Lisp Package Manager
===

### Install

**clpm** requires **sbcl**, **quicklisp** You have to install them to use clpm

``
$ sudo npm install -g clpm
```

### Usage

##### Search

default options is `-s` search

```
$ lpm restas mysql mongodb
$ lpm -s xml json
```
